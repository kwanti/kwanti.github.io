User Manual

This user manual is divided into 3 parts.

The first part describes about how to define a remote interface and the behavior of a remote method. In other words, syntax and semantics of remote interface. The chapters below belong to this part:

  • Remote Interface
  • Remote Method
  • Parameter
  • Exception & Error

The second part describes public interface for core module. It is the reference for public classes and functions of core module. The chapters below belong to this part:

  • Pool Node
  • Session
  • Server
  • Attribute
  • Io Runner
  • Global

The last part describes the communication module and the infrastructures included in this library. The chapters below belong to this part:

  • Bindings
  • Bindings Interface
  • Bindings Template
  • Bindings Factory
  • Bindings Serializer
  • Bindings Transport
